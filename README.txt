##PROJETO 2 - WISE TRAINEE 6.0

Nome do projeto: Exterminador de Lendas
Resumo: Um jogo de RPG com lendas brasileiras, usando FastAPI no back-end e Angular no front-end.

##Pré-requisitos

Antes de rodar o projeto, certifique-se de ter instalado:

-Node.js (versão LTS recomendada)

-npm (geralmente já vem junto com o Node.js)

-Angular CLI: npm install -g @angular/cli

(Obs.: Para o back-end, será necessário também Python 3.8+ MongoDB)

##Para rodar

cd LastCommit/src/frontend
npm install
ng serve
http://localhost:4200 abrir com navegador
